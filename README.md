# E.h
A C header that includes preprocessors defining everything to the letter "E" for better readability.

### Setup
To set up the program, download `e.h` and place it in the directory of your C program. Alternatively, copy all of the code from `e.h` and paste it into your own file.

### Usage
This header file uses preprocessors to redefine some basic syntax in C to the letter "E" for optimal readability. Future versions may convert everything.

The following (self explanatory due to how readable the code is) prints "E" to the console, the equivalent of a Hello World program in C without this header.
```c
#include <stdio.h>
#include "e.h"

E EE EEeee Eeeee EEeeee EEe
	e EEeee "E\n" EEeeee;

	EEE EEEe;
EEee
```

The example below shows adding two integers and printing the result to the terminal.
```c
#include <stdio.h>
#include "e.h"

E EE EEeee Eeeee EEeeee EEe
	E ee EEeeeee EEEeeeeeeee;
	E eee EEeeeee EEEeeeeeeeeee;
	E eeee EEeeeee ee EEeeeeee eee;

	e EEeee "%d + %d = %d\n", ee, eee, eeee EEeeee;

	EEE EEEe;
EEee
```
